A Locator Web-Map: Is the Sun up there?

  • This map is created to show the location of the user based on the locational information provided by the user's device.

  • The map automatically places a marker on the current user's location once the user agrees to allow the map to track the location.

  • The accuracy of your location depends on your device.

  • Please see below to learn about the key functions of this map.

    • Key Functions:

    1. Click on the star button on the left-top corner to find your location.

    2. If the Sun is up at your current location, the basemap will automatically set as a grey basemap.

    3. If the Sun has already been set, the basemap will set as a dark basemap.
